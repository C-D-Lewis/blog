As an experiment, I downloaded the Subway Subcard Android app and extracted the QR code from a screenshot. The screenshot has been modified to prevent anyone claiming my Subcard points (of which I currently have very few, but it's the principle!)

I then created a simple Pebble app to show this code as a BitmapLayer for the store scanner to scan. It looks like this with the dummy QR code: And it worked! The points were added in the Android app but I didn't have to unlock my phone and open the Subcard app. So some time was saved, and I got a positive reaction from the cashier, which is always a bonus! If you want to use this yourself, get the code from the GitHub repo and replace qr.png with a 144x144 crop of your own Subcard QR code from the app or carefully from a photo of your physical card, then re-compile.
