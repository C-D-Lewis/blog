I have another at work showing the status of the last link check on our ReadMe.io site. And now I have one as a dynamic backlight for my new PC build. And the best part? This last one has an API! It has five modes, powered by a Node.js Express server and the node-blinktÂ NPM package:
  • /set { "to": [r, g, b] } - Set a color instantly.
  • /fade { "to": [r, g, b] } - Fade from the current colour to this one.
  • 'CPU' - Fade to a colour on a HSV scale from green to red depending on current CPU load.
  • 'Screen' - Take an average of the four screen corners, and set to that colour ten times a second.
  • 'Demo' - Fade to a random colour from the rainbow every 15 seconds.
  • 'Test' - Ping the Pi and set the 'Test' button to green if it responds 'OK' and HTTP 200.
The last three are driven by a Java control panel that permanently lives on the new PC. Thanks to the motherboard supplying power after the PC turns off, I can use 'Demo' as a nightlight! Not that I need one...
